เทคโนโลยีที่ใช้

  • React

  • Styled-components

DigitalOcean App Platform เป็นบริการใหม่ของ DigitalOcean ทำให้ Build & Deploy แอพของเราง่ายขึ้น เป็นส่วนหนึ่งของการแข่งขันนี้

จุดเริ่มต้น/แรงบันดาลใจ

โปรเจคนี้สร้างเพื่อส่งเข้าแข่ง DigitalOcean Hackathon ในตอนนั้นมีความตั้งใจที่จะลองใช้ React + Apollo + MongoDB ด้วย

จุดเริ่มต้นคือ พักหลังมารู้สึกว่า Social Network ปัจจุบันไม่ค่อยเป็นมิตรเท่าไหร่ ผมเลยสร้างเว็บแอพนี้ขึ้นมาเพื่อแก้ไขปัญหานั้น

โดยข้อแตกต่างที่สำคัญที่สุดคือ ในบริการนี้จะเป็นนิรนาม ไม่จำเป็นต้องลงชื่อก็สามารถใช้ได้เลย และอีกความสามารถนึงคือ ส่งข้อความแบบสุ่ม ๆ ซึ่งจะมีเพียงคนเดียวที่ได้รับ และนั่นเป็นที่มาของแอพนี้ครับ

กระบวนการพัฒนา/อุปสรรค

จุดที่ยากที่สุดของ Project นี้คือการทำ Pagination ทั้งในส่วนความคิดเห็นและการดึงข้อมูลโพสต์ใหม่ล่าสุด รวมถึงการ Caching ด้วย เพราะกว่าจะทำให้ใช้งานได้สมบูรณ์จริง ๆ ต้องมี Boilerplate เยอะมาก

ขั้นตอนในการ Deploy ค่อนข้างจะงง ๆ หน่อยสำหรับแรก ๆ เพราะเป็นบริการใหม่ของ DigitalOcean ยังมีข้อจำกัดอยู่บ้าง

สิ่งที่ได้เรียนรู้

ได้เรียนรู้การใช้งาน Apollo, MongoDB การใช้งาน React + Apollo การใช้งาน Theme API ของ Styled-components เพื่อทำโหมด Light/Dark ฝึกการใช้งาน DigitalOcean
